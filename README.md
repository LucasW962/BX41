## BX41 Integrity Mode
BX41 includes an optional packet integrity mode that adds a compact authentication envelope:

- `u16` rolling counter (2 bytes)
- truncated `HMAC-SHA256` tag (default 4 bytes)

Default overhead is 6 bytes per payload buffer, and offsets remain encoded as internal 1-byte buffers.

### Usage

```luau
local BX41 = require(path.to.BX41)

-- Disable integrity (default)
BX41.setIntegrity(nil)

-- Enable integrity with a shared secret and default 4-byte tag
BX41.setIntegrity("my-shared-secret")

-- Enable integrity with a custom tag size (2..16 bytes)
BX41.setIntegrity("my-shared-secret", 6)
```

### Notes

- Integrity mode is optional and fully backward-compatible when disabled.
- It helps detect tampering/replay within the BX41 protocol stream.
- As with all Roblox client code, secrets present on the client are not truly private against determined exploiters.
